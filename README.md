Búsqueda Binaria: La complejidad es O(log n). Es un algoritmo muy rápido porque 
en cada paso descarta la mitad de la lista, aunque requiere que las canciones estén 
ordenadas por título para funcionar.

Playlist Automática: La complejidad es O(n). El tiempo que tarda depende directamente 
de la cantidad de canciones, ya que el código las recorre una por una sumando su duración 
hasta llegar al límite de tiempo.
